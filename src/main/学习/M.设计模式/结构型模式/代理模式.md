# 什么是代理模式

定义：为其他对象提供一种代理以控制对这个对象的访问

# 静态代理

静态代理是指预先确定了代理与被代理者的关系，就是指代理类与被代理类的依赖关系在**编译期间**就确定了。

# 动态代理

这个关系是在**运行时**确定的

目前我们代码中，基本上都是动态代理

常见的有2种

+ Cglib
+ JDK
